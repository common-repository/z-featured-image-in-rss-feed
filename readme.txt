This plugin adds featured image to your rss feed with CSS set to "display: none"

== Description ==

This Plugin adds featured image to your rss feed. It is a modification of http://wordpress.org/extend/plugins/featured-image-in-rss-feed/ set to load last and have its CSS set to "display: none". This is so it can be extracted by certain feed readers but not conflict with other image plugins

You can changes the image size from the setting page of the plugin under Settings > Image In RSS Feed. By default the plugin uses thumbnail size.

Please report any bugs @ http://dineshkarki.com.np/plugins/featured-image-in-rss-feed/report-bugs.

Thanks to <a href="http://wework4web.com" target="_blank">wework4web</a> for supporting plugin developement.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Change the image size from the setting page of the plugin under settings > Image In RSS Feed (If Required).

== Frequently Asked Questions ==

Yes, you can change the image size from the settings page of the plugin.
